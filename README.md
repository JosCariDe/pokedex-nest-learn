# Ejecutar el desarrollo

1. Clonar el Repositorio
2. Ejecutar el comando de 
```
npm install
```
3. Tener instalado Nest CLI instalado
```
npm i -g @nest/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```

## Stack Usado
* MongoDB
* Nest
